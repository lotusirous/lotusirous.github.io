Useful commands for offline installation

January 14, 2017
Tips

A collections of command for installing offline packages

Download and install packages for REHEL distribution.

The following command prepares a folder with installation packages

yum install --downloadonly --downloaddir=<directory> <package>
yum --disablerepo=* localinstall *.rpm
Install python package from local.

The following command installs keras package from local

pip install --no-index --find-links . keras
